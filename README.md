## Запуск проекта локально
Необходимо запустить pgAdmin и создать в нём базу данных с названием 'rlt_test'
Если данные в pgAdmin и данные к подключению в проекте разные, то измените параметры метода psycopg2.connect([параметры]) во всех файлах.
Далее в командной строке введите 'py seed.py'. Это вставит данные из videos.json в две таблицы в базу данных.
И наконец, для проверки работы бота, в командной строке наберите 'py bot.py'.
## Как задать токен Telegram-бота
В файле bot.py есть постоянная TOKEN в 8-ой строке файла. Там и необходимо задать токен.
## Краткое описание архитектуры и подхода к преобразованию текстовых запросов в обращения к БД
Для ввода данных из файла json.py я создал запрос 'CREATE TABLE' для двух таблиц. Затем с помощью for-циклов вводил запросы 'INSERT INTO' для заполнения данными таблицы.
## Использование LLM
Я вставил текст пользовательского запроса в общий запрос в ИИ, при этом подробно описав структуру полей в таблицах.
